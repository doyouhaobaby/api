Entity

abstract class Entity implements IEntity, IArray, IJson, JsonSerializable, ArrayAccess

模型实体 Object Relational Mapping.

为最大化避免 getter setter 属性与系统冲突 系统自身的属性均加前缀 leevel，设置以 with 开头, 获取数据不带 get. 包含 Leevel\Database\Connect[包含继承], Leevel\Database\Select. ORM 主要基于妖怪的 QeePHP V2 设计灵感. 例外参照了 Laravel 关联模型实现设计. Doctrine 和 Java Hibernate 中关于 getter setter 的设计
